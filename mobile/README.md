# Route Intelligence Mobile App (Android)

Android mobile application for landscape operators to manage routes, view jobs, and navigate optimized routes on the go.

## Features

- Route navigation
- Job management
- Real-time route updates
- Offline capabilities
- Push notifications

## Tech Stack

- React Native or Native Android (Kotlin/Java)
- Mapbox SDK for Android
- Room Database for offline storage

## Development

### Prerequisites

- Android Studio
- JDK 11+
- Android SDK

### Setup

1. Open project in Android Studio
2. Sync Gradle files
3. Configure API endpoint in `Config.kt`
4. Run on device or emulator

## Build

```bash
./gradlew assembleRelease
```
